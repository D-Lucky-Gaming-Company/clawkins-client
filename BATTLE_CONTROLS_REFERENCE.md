# Battle Controls Quick Reference

## Keyboard Controls

### Battle Actions
```
[1] → Attack
[2] → Defend  
[3] → Special
[4] → Item
```

### System Controls
```
[E] → Toggle Inventory (Open/Close)
[X] → Attempt to Run (Shows confirmation)
[Z] / [Space] / [Enter] → Confirm / Advance Dialogue
[Escape] → Cancel Run Confirmation
```

---

## Run Confirmation Flow

1. Press **[X]** during battle
2. Prompt appears: "Are you sure you want to run?"
3. Choose:
   - **[Z]** → Yes, attempt to run
   - **[X]** → No, return to battle

**Note**: You can only run from wild battles, not trainer battles.

---

## Inventory System

- Press **[E]** to open inventory from battle
- Press **[E]** again to close and return to battle
- Battle state is preserved while inventory is open
- Can be toggled repeatedly without issues

---

## Visual Indicators

### Clawkin Container (Left Side)
- **Top Slot** → Ginger
- **Middle Slot** → Swee'pea
- **Bottom Slot** → Dart

### Selection Highlight
- **Brighter icon** = Currently active Clawkin
- **Normal icon** = Inactive party member
- **Grayed icon** = Knocked out (0 HP)

The highlight automatically updates when you switch Clawkins.

---

## Important Notes

- **No Mouse Input**: All battle interactions use keyboard only
- **Buttons are Visual**: On-screen buttons show key bindings but don't respond to clicks
- **Confirmation Required**: Running always requires confirmation to prevent accidents
- **Wild Battles Only**: Flee button is disabled in trainer battles
